# Lizard

This is WIP online multiplayer card game, built with the [boardgame.io](https://boardgame.io) framework.

Chek RULES.md for more information how the game is played.

This project is WIP and this README will be updated once the game is in a fully playable state.

## Development

### Start client

`npm run client`


### Start server

`npm run server`


### Run Tests

`npm run test`

Run tests with coverage:

`npm run test -- --coverage`
